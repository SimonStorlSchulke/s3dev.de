FUDGE

TypeScript, WebGL, HTML, CSS

Furtwangen University Didactic Game Editor is a TypeScript based GameEngine especially for the teaching of Prof. Jirka Dell’Oro-Friedl. Here I worked on the implementation of mesh generation, OBJ mesh import and shaders. In the Chemistry with FUDGE project, we used the engine in very early stages to implement an atom orbital construction kit.
